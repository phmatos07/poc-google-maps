Nesse módulo você terá acesso uma demonstração pratica da utilização das principais funções da API do Google Maps.

Acessar Map Marker

O componente MapMarker exibe um marcador no mapa quando é um filho de conteúdo de um componente GoogleMap. Como o GoogleMap, este componente oferece uma entrada de opções, bem como entradas de conveniência para posição, título, rótulo e clicável, e oferece suporte a todos os eventos google.maps.Marker como saídas.
